# Image Upload Guide

## Folder Structure

All your project images should be placed in:
```
images/projects/
```

## File Naming Convention

For **Project 1**, you need **TWO** files:

### 1. Thumbnail Image (Left Column)
- **File name**: `project-1-thumb.jpg` (or `.png`, `.webp`)
- **Location**: `images/projects/project-1-thumb.jpg`
- **Recommended size**: 150x150px to 300x300px (square)
- **File size**: Keep it small (< 100KB) for fast loading

### 2. Full-Size Image (Center Display)
- **File name**: `project-1.jpg` (or `.png`, `.webp`)
- **Location**: `images/projects/project-1.jpg`
- **Recommended size**: 600x600px or larger (maintains aspect ratio)
- **File size**: Optimized, but can be larger than thumbnail

## For Other Projects

Follow the same pattern:
- Project 2: `project-2-thumb.jpg` and `project-2.jpg`
- Project 3: `project-3-thumb.jpg` and `project-3.jpg`
- ... and so on

## Supported Image Formats

- `.jpg` / `.jpeg` (recommended for photos)
- `.png` (good for graphics with transparency)
- `.webp` (modern, smaller file size)

## Tips

1. **Keep file names lowercase** with hyphens (dashes) between words
2. **No spaces** in file names (use hyphens instead)
3. **Use consistent naming**: `project-X-thumb.jpg` and `project-X.jpg`
4. **Optimize images** before uploading to reduce file size and improve loading speed
5. You can use the same image for both thumbnail and full-size if needed (just name them differently)

## Quick Steps for Project 1

1. Prepare your image(s)
2. If you have one image:
   - Copy it twice
   - Resize one to ~150x150px and name it `project-1-thumb.jpg`
   - Keep the full-size version and name it `project-1.jpg`
3. Place both files in: `/images/projects/`
4. The website will automatically use them!
